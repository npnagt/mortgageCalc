# Mortgage Calculator

A simple React mortgage calculator built with Vite.

## Features

- Principal amount input
- Down payment percentage input
- Interest rate input
- Loan duration in years
- Monthly payment, total payment, and total interest results

## Run locally

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the development server

   ```bash
   npm run dev
   ```

## Deploy on Vercel

This project is ready for Vercel deployment.
Just connect the repository and use the default build settings. The build command is `npm run build` and the output directory is `dist`.
